Hi and welcome to TephraProb! To use it, navigate to the root of the TephraProb 
folder in Matlab and type in the Matlab command:
>> tephraProb
and press enter. Refer to the user manual for detailed decriptions of the 
functionalities of the code. A video tutorial is also available:
https://www.youtube.com/channel/UCP8gCjSeMoPVwgzMwKUnW3w

TephraProb is released under a GPL3 license, which means that everybody should 
feel free to contribute, comment, suggest and modify the code for as long as any 
new update remains open-source. Don't hesitate to contact me by email should you 
have a suggestion or find a bug.

Hope this code will help!
